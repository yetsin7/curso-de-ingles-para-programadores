## 🇳🇮 Para programadores de Nicaragua (y de toda Latinoamérica)

Si estás leyendo esto desde Nicaragua, este libro fue pensado especialmente para ti.

Aquí en Nicaragua tenemos talento de sobra. Lo que muchas veces nos falta no es capacidad técnica, sino una herramienta concreta: **el inglés técnico**. Es la barrera número uno que detiene a programadores nicaragüenses talentosos cuando intentan leer documentación oficial, ver tutoriales avanzados, aplicar a trabajos remotos o pasar entrevistas técnicas internacionales.

Este libro existe para tumbar esa barrera. No necesitas pagar cursos caros, no necesitas registrarte en ninguna plataforma y no necesitas internet constante. Una vez que clones el repositorio, puedes estudiar offline desde donde estés.

Si eres autodidacta, estudiante, recién graduado o ya llevas tiempo programando pero el inglés sigue siendo tu punto débil, este material es para ti.

---

## 🎯 ¿Por qué existe este libro?

La misión es simple y directa: **que más nicaragüenses dominen el inglés técnico necesario para crecer profesionalmente como programadores**.

Dominar el inglés técnico es una de las mayores diferencias entre un programador que se queda estancado y uno que avanza. Te abre las puertas a:

- Leer documentación oficial sin depender de traducciones a medias.
- Entender mensajes de error y resolver problemas más rápido.
- Aplicar a trabajos remotos pagados en dólares.
- Pasar entrevistas técnicas en inglés con confianza.
- Aprender de los mejores tutoriales, libros y conferencias del mundo.
- Contribuir a proyectos open source internacionales.

Este recurso es **100% gratuito**, **no requiere registro** y se puede estudiar **completamente offline** una vez clonado el repositorio. La única inversión es tu tiempo y disciplina.

---

## 📖 ¿Qué es esto?

El **Curso de Inglés para Programadores** es una colección de lecciones organizadas como un libro técnico interactivo. Su meta es que puedas leer, escuchar, pronunciar y escribir inglés técnico con confianza real.

- Explicaciones detalladas en **español**
- Vocabulario técnico real del mundo de la programación
- Pronunciación con guías fonéticas
- Ejemplos con código real
- Ejercicios prácticos con soluciones

Está diseñado para acompañar al **Curso de Flutter**, **Curso de JavaScript** y **Curso de Python** del mismo autor.

---

## 👥 ¿Para quién es?

Para **cualquier programador hispanohablante** que quiera:

- Aprender inglés desde cero con contexto de programación
- Leer documentación técnica en inglés con confianza
- Escribir código, comentarios y documentación en inglés profesional
- Comunicarse en equipos internacionales de desarrollo
- Prepararse para entrevistas técnicas en inglés

No se requiere experiencia previa con inglés. Solo ganas de aprender y práctica constante.

## 🧭 ¿Cómo usar este libro?

1. **Ve en orden** — cada capítulo construye sobre el anterior.
2. **Lee las explicaciones** — están en español para que entiendas todo.
3. **Practica la pronunciación** — usa las guías fonéticas incluidas.
4. **Haz los ejercicios** — están al final de cada lección.
5. **Aplícalo al código** — cada lección tiene ejemplos con código real.

> 💡 Consejo: estudia un poco todos los días, aunque sean 20 minutos. La constancia gana siempre.

## 💻 Cómo empezar (offline)

Solo necesitas Git instalado. Una vez clonado, puedes estudiar sin conexión.

```bash

cd Curso-de-Ingles-para-programadores
```

Abre los archivos con tu editor favorito (VS Code, por ejemplo) o léelos directamente desde GitHub.
