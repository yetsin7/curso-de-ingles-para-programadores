# Recursos de referencia para ingles para programadores

Este curso no depende de una sola documentacion oficial, pero si conviene usar
fuentes confiables para vocabulario, escritura tecnica y pronunciacion.

## Referencias principales

- [Cambridge Dictionary](https://dictionary.cambridge.org/)
- [Oxford Learner's Dictionaries](https://www.oxfordlearnersdictionaries.com/)
- [Merriam-Webster Dictionary](https://www.merriam-webster.com/)
- [Google developer documentation style guide](https://developers.google.com/style)

## Recursos clave para estudiar

- [Guia de pronunciacion en Cambridge](https://dictionary.cambridge.org/help/phonetics.html)
- [Palabras mas comunes en ingles](https://www.oxfordlearnersdictionaries.com/wordlists/oxford3000-5000)
- [GitHub Docs style guide](https://docs.github.com/en/contributing/style-guide-and-content-model/style-guide)
- [MDN Glossary](https://developer.mozilla.org/en-US/docs/Glossary)

## Orden recomendado

1. Refuerza vocabulario general y pronunciacion.
2. Aprende verbos y estructuras frecuentes en tecnologia.
3. Lee documentacion real en ingles todos los dias.
4. Practica escribir mensajes tecnicos y explicaciones cortas.
